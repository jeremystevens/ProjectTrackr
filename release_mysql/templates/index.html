About FlaskBin

FlaskBin is a modern pastebin application built with Flask. It allows you to share code snippets and text notes quickly and easily.

Features:
  • Syntax highlighting for various languages
  • Public and private pastes
  • Optional expiration
  • Burn after read option
  • User accounts for tracking your pastes
